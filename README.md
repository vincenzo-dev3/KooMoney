## ğŸ£ What is Koo?

**Koo Money** is a social payment protocol designed to solve Hedera's "First Mile" problem: **User Onboarding.**

Currently, gifting or sending crypto to a new user requires the recipient to have a wallet, a seed phrase, and an exchange account setup beforehand. This friction kills adoption.

**Koo reverses the flow:**
1.  **Asset First:** Wrap HBAR, USDC, or HTS Tokens into a "Magic Link" (Smart Escrow).
2.  **Wallet Second:** Send the link via WhatsApp, Twitter, or Email. The recipient claims it using **Google Login**, creating a non-custodial Hedera wallet instantly.

> *"We are building the infrastructure to make crypto transactions as seamless as sending a text message."*
